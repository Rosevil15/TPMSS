# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-28

### Added
- Initial release of L.E.A.P platform
- User authentication with email/password and Google OAuth
- Role-based access control (Admin, Health Worker, Social Worker, School, User)
- Health monitoring system (prenatal, postnatal, child health records)
- Case management functionality
- Profile management system
- Early warning dashboard
- Report generation (PDF, Excel)
- Location-based filtering (Philippine regions, provinces, municipalities, barangays)
- Privacy agreement enforcement
- Comprehensive test suite (unit and e2e)

### Security
- Environment variable configuration for sensitive data
- Supabase Row Level Security implementation
- Secure OAuth integration
